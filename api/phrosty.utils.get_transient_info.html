get_transient_info¶

phrosty.utils.get_transient_info(oid)[source]¶

Retrieve RA, Dec, start MJD, and end MJD for a specified object in the OpenUnvierse sims.

This function calls get_transient_radec() and get_transient_mjd().

Parameters:

oid (int) – Object ID of target transient.

Returns:

  • ra (float) – RA in degrees of target transient.

  • dec (float) – Dec in degrees of target transient.

  • start (float) – Simulated start MJD of target transient.

  • end (float) – Simulated end MJD of target transient.

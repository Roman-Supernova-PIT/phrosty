pointings_near_mjd¶

phrosty.utils.pointings_near_mjd(mjd, window=3, obseq_path=None)[source]¶

Retrieve pointings near given MJD.

Parameters:

  • mjd (float) – Central MJD to search around.

  • window (float, default 3) – Number of days around central MJD to include in search.

  • obseq_path (Path, default None) – Path to obseq file Roman_TDS_obseq_11_6_23.fits.

Returns:

pointings – List of pointings within specified MJD range.

Return type:

list

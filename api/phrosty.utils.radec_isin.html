radec_isin¶

phrosty.utils.radec_isin(ra, dec, path=None, band=None, pointing=None, sca=None)[source]¶

Check if a given RA, dec coordinate is in a given target image.

Parameters:

  • ra (float) – RA in degrees.

  • dec (float) – Dec in degrees.

  • path (Path, default None) – Path to image to check.

  • band (str, default None) – Filter assocated with target image.

  • pointing (str, default None) – Pointing associated with target image.

  • sca (str, default None) – SCA associated with target image.

Returns:

res – True if provided RA, Dec is in the image. False if not.

Return type:

boolean
